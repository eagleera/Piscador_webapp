# piscador_webapp

## Para instalar el proyecto
```
npm install
```

### Para correrlo
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files

npm run lint
